Formatação de textos, Cores e Download de fontes

04/02/2025

Existem diversas formas de se formatar textos com o HTML. Para que seja possível fazer isso, é necessário o uso de tags envolvendo a área em que você queira aplicar a formatação. Por exemplo, caso você queira deixar um texto em negrito, deve se escrever o código desta forma: "<strong>Texto em negrito</strong>". Mais exemplos de tags de formatação no site W3Schools.

No CSS, é usado o padrão RGB para se definir cores. Para se fazer isso, pode-se usar o nome , valores hexadecimais, decimais ou percentuais da cor. Por exemplo, "blue", "#0000FF", e "rgb(0,0,255)" representam a cor azul de formas diferentes. Existem 140 nomes de cores que podem ser utilizados como valores para cores. Apesar da enorme quantidade, para que você utilize cores mais personalizadas, é recomendado definir cores através de outras formas, como por exemplo, usar seu código hexadecimal ou rgb. Um pouco mais sobre as cores do CSS no site W3Schools.

O CSS oferece algumas fontes padrão para adicionar à sua página html, mas você sabia que é possível utilizar várias outras fontes? O Google oferece um serviço para baixar fontes via CSS. Esse serviço disponibiliza além das fontes, o CSS necessário para usá-las. Para aprender mais sobre o download e aplicação de fontes, acesse o Google Fonts.

Imagens e Links

07/02/2025

Por volta da década de 90, imagens foram introduzidas às páginas na internet. As imagens, mesmo que possuissem a menor resolução possível, aumentavam o tempo de carregamento das páginas. Existem diversos formatos de imagens, como por exemplo, os formatos GIF, JPEG, BMP, PNG, TIFF, SVG, ICO e WebP. Para se adicionar uma imagem a sua página HTML, é utilizada a tag <img>. Exemplo: <img src = "FONTE DA IMAGEM" alt = "DESCRIÇÃO DA IMAGEM">. "src" é um elemento obrigatório e representa a fonte da imagem, seja essa um link para a imagem ou a pasta em que a imagem se localiza em seu computador. Já o elemento "alt", também obrigatório, apresenta um texto de descrição da imagem. Para aprender pouco mais sobre as imagens, acesse o site W3Schools.

Como você já deve ter visto nas postagens anteriores, é possível criar hiperlinks no HTML. Os links são úteis para adicionar fontes do conteúdo, informações adicionais ou importantes, além de também serem utilizados para levar o usuário a outras partes de uma página ou website. Para se adicionar um hiperlink, é usada a tag <a>. Exemplo: <a href="url">TEXTO DO LINK</a>. Para aprender mais sobre os links, acesse o site W3Schools.

Listas enumeradas e não enumeradas

08/02/2025

No HTML, listas são tags que permitem agrupar itens de uma forma estruturada. Basicamente, há uma tag para definição e outra para os itens da lista. Entre os tipos de listas, temos as listas ordenadas e listas não ordenadas. Listas ordenadas são usadas para representar itens com enumerações ou sequências. Elas podem seguir uma ordem numérica, alfabética ou em algarismos romanos. Para se fazer uma lista ordenada, é usada a tag <ol> (Ordered List) envolvendo os itens <li> (List Item). O atributo "start" serve para começar a numeração em um número/letra diferente. Já o atributo "reversed" inverte a ordem da numeração. Para saber um pouco mais sobre o funcionamento das listas ordenadas, acesse o site W3Schools.

As listas não ordenadas são utilizadas para representar itens não enumerados. A formatação dessas listas se dá por símbolos/bullets ou imagens.Usa-se uma tag <ul> (Unordered List) envolvendo os itens <li> (List Item). O padrão de formatação é o “disc”. Para saber mais sobre as listas não ordenadas, acesse o site W3Schools.

Tabelas

09/02/2025

As tabelas são construções em HTML que permitem agrupar texto e outros conteúdos (até outras tabelas). Elas são tipicamente usadas para exibir listagens ou registros em linhas e colunas. Os elementos principais de uma tabela são os cabeçalhos (table headers), linhas e colunas formando as células, eventualmente linhas ou colunas expandidas e bordas. Para as tags de uma tabela, são usadas três tags duplas: table (tabela), tr (table row), e td (table data). De forma hierárquica, as tabelas possuem linhas que possuem colunas.

<table> define a tabela, <tr> cria uma linha ou mais linhas dentro de <table> e <td> define uma ou mais colunas dentro de <tr>. A tag <th> (table header) é uma coluna que define um cabeçalho para a tabela. A tag <th> é também semântica e dá um destaque ao texto. Os atributos "colspan" e "rowspan" são utilizados para expandir linhas e colunas. Esses atributos definem que uma coluna ou linha ocupam mais que seu espaço original. Para aprender mais sobre as tabelas do HTML, acesse o site W3Schools.
